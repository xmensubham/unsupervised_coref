\section{Overview}
% ========================================================================

Phase~2 extends the unsupervised coreference resolution pipeline
introduced in Phase~1 with two key enhancements:

\begin{enumerate}
    \item \textbf{Span-based mention representation:} Each mention is
          represented by its character-level span in the document, enabling
          finer-grained tracking of coreferent expressions.
    \item \textbf{Probabilistic link decisions:} A \texttt{resolve\_text}
          function assigns each anaphor a \textit{coreference probability}
          (\texttt{p\_coref}) with respect to its best antecedent. A
          configurable threshold (default $\tau = 0.55$) determines whether
          a link is accepted (\texttt{linked = True}) or suppressed
          (\texttt{linked = False}).
\end{enumerate}

The Phase~2 system is evaluated on the AI4Bharat Hindi coreference
dataset using the standard MUC metric, and its behaviour is illustrated
with a worked example on a three-sentence Hindi passage.
